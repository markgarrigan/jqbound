## Up and Running

Use Vagrant https://www.vagrantup.com/

```bash
cd /path/to/mockingj
vagrant up
```

Visit localhost:4141 in your browser.

### or

Serve up the dist directory however you'd like.

## Build

```
git clone
cd /path/to/ediblecontent
npm install
gulp
```

Make changes to files in src.
